Source Code on GitHub

The full material to produce this book via Jupyter Book Community (2020), except for some of the used datasets, can be found on the GitHub repository. Very small datasets were included in the GitHub repository, but anything slightly larger is freely available elsewhere and links are provided in the book and the provided code (usually facilitating automatic downloads when necessary).

Contribute¶

We welcome feedback, comments and contributions to this book. For mistakes, issues etc. please open an issue on GitHub, or comment to an existing one. Thanks!
